如何运行 ZfDash 代理

要管理远程主机，请在该机器上运行 ZfDash 代理:

如果已安装 ZfDash:
sudo zfdash --agent
从源码运行:
sudo python src/main.py --agent
从源码完整安装:
git clone https://github.com/Jioyzen/zfdash-zh-cn.git
cd zfdash-zh-cn
uv sync
sudo .venv/bin/python src/main.py --agent
注意: 代理默认监听 5555 端口并启用 TLS。 首次运行时，系统会提示您设置管理员密码。
